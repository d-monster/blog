django强行入门——链接mysql数据库

最近纵向搭个网站，于是找了django框架来学习，然而python已经忘得差不多了，之前又花了好几天把python复习了一下，写了几个爬虫练练手，今天终于可以开始操作了。

创建数据库

比如创建一个叫做mysite的数据库

更改配置

  • 首先需要更改settings.py

    找到DATABASES，改成

    DATABASES = {
'default': {
'ENGINE': 'django.db.backends.mysql',
'NAME': 'db_name',        //数据库的名字
'HOST': '',              //主机名，自己学习用一般是localhost
'PORT': '3306',             //端口，数据库服务的端口，mysql默认是3306
'USER': 'root',             //数据库的用户名，一般是要root用户，权限才够
'PASSWORD': '41223',      //root对应的密码
    }
}

  • 更改models.py

    创建表对应的类，例如：

    class Publisher(models.Model):        //出版商类，对应mysite数据库中的一个表
    pub_name = models.CharField(max_length=200)
    city = models.CharField(max_length=200)

class Book(models.Model):            //书类
    book_name = models.CharField(max_length=200)
    author = models.CharField(max_length=200)
    pub_date = models.DateTimeField('date published')
    publish = models.ForeignKey(Publisher, on_delete=models.CASCADE)
